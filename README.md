# ModShield AI - Discord Moderation Bot

AI-powered moderation for Discord servers with context awareness, mod burnout prevention, and raid detection.

## Quick Start

1. Download `modshield_ai_prototype.html`
2. Open in browser (works offline)
3. Try the Message Simulator tab
4. Read `MODSHIELD_AI_REFACTORED_PLAN.md` for full instructions
